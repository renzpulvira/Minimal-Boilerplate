# A Minimal Web ready boilerplate

### Requirements
1. Nodejs
2. Npm

### What's inside
1. Bootstrap 4.
2. Fontawesome 5
3. A gulpfile for 
   crossbrowser prefixes
   and browsersync.
4. SMACCS ready file structure.
5. Useful mixins.

### How to use
1. Open your terminal and cd to the folder.
2. inside the folder run `npm install`.
3. run `gulp watch`.

> THATS IT!, HAPPY CODING !
